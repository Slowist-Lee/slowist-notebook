    1 实验目的和要求¶

    1.1 实验目的¶

    （1）构建一个信号。 （2）分析其频谱，并对频谱作相应处理。 （3）观察信号的变化,探究采样频率、采样点数对信号的影响。

    1.2 实验要求¶

    （1）构建一个信号，至少含有 3 个频率分量，可以是周期信号，也可以是非周期信号（如加随机噪声)。 （2）利用 MATLAB 分析其频谱，并对频谱作滤波和增加频率成分的处理 （3）观察信号的变化,探究采样频率、采样点数对信号的影响。

    2 实验原理¶

    1. 物理原理：较高频率信号的波长较短，干扰源对波长较短的信号的影响相对较小。
    2. 傅里叶变换： 对于非正弦连续周期信号，如果满足狄里赫利条件，都可以用傅里叶级数分解为无穷多个谐波分量的叠加，即x(t)=x(t+mT)=∑k=0+∞akejkω0t →X(ω)=∫−∞+∞x(t)ejkωtdt=∫0∞e−αtejkωtdt=1α+jω
    3. 使用matlab实现傅里叶变换和逆变换：

    傅里叶变换的语法：

    C
    1
    Y = fft(X) %用快速傅里叶变换（FFT）算法计算 X 的离散傅里叶变换（DFT）。

    • 如果 X 是向量，则 fft(X) 返回该向量的傅里叶变换。
    • 如果 X 是矩阵，则 fft(X) 将 X 的各列视为向量，并返回每列的傅里叶变换。
    • 如果 X 是一个多维数组，则 fft(X) 将沿大小不等于 1 的第一个数组维度的值视为向量， 并返回每个向量的傅里叶变换。

    c Y = fft(X,n) %说明：返回 n 点 DFT。如果未指定任何值，则 Y 的大小与 X 相同。

    • 如果 X 是向量且 X 的长度小于 n，则为 X 补上尾零以达到长度 n。
    • 如果 X 是向量且 X 的长度大于 n，则对 X 进行截断以达到长度 n。
    • 如果 X 是矩阵，则每列的处理与在向量情况下相同。
    • 如果 X 为多维数组，则大小不等于 1 的第一个数组维度的处理与在向量情况下相同。

    傅里叶逆变换的语法：

    C
    1
    X = ifft(Y) %说明：使用快速傅里叶变换算法计算 Y 的逆离散傅里叶变换。X 与 Y 的大小相同。
    • 如果 Y 是向量，则 ifft(Y) 返回该向量的逆变换。 • 如果 Y 是矩阵，则 ifft(Y) 返回该矩阵每一列的逆变换。
    • 如果 Y 是多维数组，则 ifft(Y) 将大小不等于 1 的第一个维度上的值视为向量，并返回 每个向量的逆变换。
    C
    1
    X = ifft(Y,n) %说明：通过用尾随零填充 Y 以达到长度 n，返回 Y 的 n 点傅里叶逆变换。

    3 实验内容¶

    3.1 构建信号¶

    1. 指定信号参数：采样频率为1kHz，信号持续时间为1.5秒。
    2. 构造一个信号，其中包含幅值为 0.8 的 80 Hz 正弦量、幅值为 1.2 的 120 Hz 正弦量和幅值为 1.8 的 180 Hz 正弦量，并绘制相应图像，保存为文件 "Signal No.1.png"
    3. 用均值为零、方差为 4 的白噪声扰乱该信号,再绘制相应图像，保存为文件 "Signal+Noise No.2.png"

    3.2 对含噪信号的频谱分析¶

    1. 对含噪的信号进行频谱分析，对其做傅里叶变换，获得其幅值，并生成对应图像，保存为"Single‐Sided Amplitude Spectrum of X(t) with noise.png"

    3.2 对不含噪信号的频谱分析和相应处理¶

    1. 对含噪的信号进行频谱分析，对其做傅里叶变换，获得其幅值，并生成对应图像，保存为"Single‐Sided Amplitude Spectrum of X(t) NO noise.png"
    2. 对频谱做滤波的处理：删除其小于150Hz的信号，再进行逆傅里叶变换，生成对应图像，保存为“After signal processing.png”
    3. 在对应信号基础上，加上幅值为 1.0 的 200 Hz 正弦信号，并生成对应图像，保存为"Signal Added.png"

    3.3 探究采样频率、采样点数对信号的影响¶

    3.3.1 探究采样点数对信号的影响¶
    1. 构造一个采样频率为1kHz,幅值为 1.2 的 80 Hz 正弦信号，并绘制相应图像，保存为文件 "Signal No.1 1500 points.png"
    2. 保持采样频率为1kHz，幅值 1.2 ,频率 80 Hz 不变，将采样点个数改为50，并绘制相应图像，保存为文件 "Signal No.2. 50 points.png" 观察和比较两张图片的不同。
    3.3.2 探究采样频率对信号的影响¶
    1. 构造一个采样频率为1kHz,包含幅值为 1.2 的 80 Hz 正弦信号和幅值为0.8的80Hz的正弦信号，并绘制相应图像，保存为文件 "Signal No.3 Fs=1000.png"
    2. 将采样点个数改为200Hz，并绘制相应图像，保存为文件 "Signal No.4. Fs=200.png" 观察和比较两张图片的不同。

    5 实验结论¶

    • 在谱图的分析处理实验中，可以得出实验结论为：越高频的信号会产生更好的频率近似值，抗干扰能力强
    • 在“探究采样点数对信号的影响”实验中，得知“采样点个数”和“信号的时间分辨率”有关，较多的采样点可以提供更多的信息，更准确地表示信号；而过少的采样点会导致信号信息的丢失，使得信号的精细结构无法得到准确的表示。
    • 探究采样频率对信号的影响实验验证了奈奎斯特定理，即如果要完全避免采样导致的混叠现象，采样频率必须至少是信号频率的两倍。
